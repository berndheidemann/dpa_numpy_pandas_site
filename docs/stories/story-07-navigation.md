# Story 7 – Navigation nutzen

**Als** Nutzer **möchte ich** zwischen Suche, Favoriten und Info navigieren **damit** ich die Hauptbereiche der App erreiche.

## Akzeptanzkriterien
- Eine Navbar ist sichtbar.
- Die aktive Route ist optisch hervorgehoben.
- Ein Klick führt zu den jeweiligen Seiten.

## Tasks
- [ ] Komponente `Navbar` mit Links zu allen Hauptseiten erstellen (Suche, Favoriten, Info)
- [ ] React Router `NavLink` verwenden und aktive Route mit CSS-Klasse hervorheben
- [ ] Alle Routes im Router konfigurieren (`/`, `/favorites`, `/about`, `/city`)
- [ ] Navbar im App-Layout einbinden (über allen Pages sichtbar)
- [ ] Responsive Navigation für Mobile implementieren (optional: Burger-Menü)
