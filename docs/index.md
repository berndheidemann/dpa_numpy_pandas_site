# Die Welt von Zuul – Lernsituation

## Projektbeschreibung

In dieser Lernsituation wird das textbasierte Adventurespiel **"Die Welt von Zuul"** schrittweise erweitert und verbessert. Dabei werden grundlegende Konzepte der objektorientierten Programmierung wie **Kopplung**, **Kohäsion**, **Kapselung** und verschiedene **Design Patterns** erlernt und angewendet.

Das Ausgangsprojekt ist ein einfaches Spiel, in dem sich ein Spieler durch verschiedene Räume einer virtuellen Welt bewegen kann. Im Verlauf der Lernsituation wird dieses Projekt um neue Funktionen erweitert, wobei der Fokus auf **gutem Softwaredesign** liegt.

## Lernziele

- **Kopplung und Kohäsion** verstehen und anwenden
- **Kapselung** als Grundprinzip der OOP einsetzen
- **Refactoring** zur Verbesserung von bestehendem Code durchführen
- **Exceptions** zur Fehlerbehandlung implementieren
- **Design Patterns** kennenlernen und anwenden:
    - State Pattern
    - Command Pattern
    - Factory Pattern / Factory Method Pattern
- **Git** zur Versionsverwaltung nutzen
