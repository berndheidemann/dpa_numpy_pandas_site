# Umsetzungshilfe – Story 8: Info-Seite

## Struktur
```
src/
  pages/
    AboutPage.tsx      # Route /about mit Beschreibung
```

## Inhalte
- Erkläre kurz den Zweck des Dashboards, die Datenquelle (Open-Meteo) und die wichtigsten Funktionen (Suche, Detail, Favoriten).
- Erwähne die technische Basis (React, ggf. Bootstrap, Recharts).

## Exkurs: UX Writing
- Schreibe klar und knapp.
- Nutze Zwischenüberschriften oder Absätze für bessere Lesbarkeit.
- Achte auf ausreichenden Kontrast und eine angenehme Zeilenlänge, damit der Text auf Desktop und Mobile gut funktioniert.
- Falls vorhanden, verlinke zu Datenschutz oder Impressum.
